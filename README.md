## 📋 What is ComfortADHD?

ComfortADHD is a user-friendly software designed to help identify early signs of ADHD. It uses smart technology to analyze behavior and provide clear insights. This helps users and healthcare providers spot potential ADHD symptoms sooner.

With ComfortADHD, you don’t need technical skills to access advanced tools. The program includes a virtual chatbot that offers support and guidance, acting like a friendly assistant. It blends machine learning models with large language models to give you accurate and understandable results.

## 🔍 About This Project

ComfortADHD combines multiple machine learning techniques to offer a comprehensive tool for early ADHD screening.

- Stacking Ensemble ML models improve prediction accuracy by combining several algorithms.
- Explainable AI (XAI) helps users and clinicians understand how the software reaches its conclusions.
- Large Language Models (LLMs) analyze behavioral inputs and power an engaging virtual therapist chatbot.
- The chatbot uses Dialogflow technology to simulate supportive therapy dialogue, bridging the gap between screening and ongoing patient care.

The software is built using Python and focuses on data preprocessing, exploratory data analysis, and robust performance metrics.
